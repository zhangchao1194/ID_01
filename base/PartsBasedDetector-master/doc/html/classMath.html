Reduce a vector of matrices via indexing.

Reduce a 3D matrix (represented as a vector of matrices using cv::split() ) to a 2D matrix by choosing a single element from each ray cast through the third dimension. This loosely emulates Matlab's matrix indexing functionality

out[i][j] == in[i][j][ idx[i][j] ], forall i,j

Parameters:
inthe input 3D matrix
idxthe index to choose at each element (idx.size() == in[k].size() forall k)
outthe flatten 2D output matrix
