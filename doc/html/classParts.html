Detailed Description

a monolithic collection of part parameters

Parts describes a container class which holds a monolithic collection of part parameters. Its primary purpose is to hold concrete values of each parameter and spawn lightweight ComponentPart classes (which just have pointers to the same parameters) when object retrieval is required.

It is designed to maximally reflect the configuration of the Matlab implementation to avoid confusion
