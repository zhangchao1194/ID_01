Detailed Description

template<typename T>
class PartsBasedDetector< T >

The main object detection class PartsBasedDetector is the main entry point for detecting objects. It has a single method distributeModel() for setting up the detector parameters from a deserialized model, and a method detect() for running the detection pipeline.

Template Parameters:
Tthe detector precision. Should be one of float or double. On modern 64-bit machines, the latter will likely be just as fast.
