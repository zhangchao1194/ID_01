Calculate the responses of a set of features to a set of filter experts.

A response represents the likelihood of the part appearing at each location of the feature map. Parts are support vector machines (SVMs) represented as filters. The convolution of a filter with a feature produces a probability density function (pdf) of part location

Parameters:
featuresthe input features (at different scales, and by extension, size)
responsesthe vector of responses (pdfs) to return
